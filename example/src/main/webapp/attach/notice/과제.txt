0. 성명

1. 내PC(내컴퓨터)->마우스 오른쪽 클릭->속성
   -> 캡쳐(alt+print Screen) 
   -> 한글이나 word 문서에 붙이기(ctrl+v)

2. 내PC(내컴퓨터)->마우스 오른쪽 클릭->속성
   -> 장치관리자 클릭 
   -> 디스크드라이브 ">"기호 클릭
   -> 디스플레이 어댑터 ">"기호 클릭
   -> 캡쳐(alt+print Screen) 
   -> 한글이나 word 문서에 붙이기(ctrl+v)

3. 내PC(내컴퓨터) 더블클릭
   -> 캡쳐(alt+print Screen) 
   -> 한글이나 word 문서에 붙이기(ctrl+v)

4. 모니터 인치(inch) - 듀얼 모니터(O,X)

5. 데스크탑인지 노트북인지 작성

6. 완제품(삼성,LG,주현,한성 등)인 경우 모델명 작성

7. 대략적인 구매년도 기록(중고 여부 포함)
